# iPhone_aquarium_Prototype
水族館図鑑のiPhoneアプリのプロトタイプ．  
iPhoneにて写真を撮影すると，対象の水族館の生物を認識して、その生物に関する情報を表示する．
現状は`イルカ，クラゲ，カクレクマノミ，カワウソ，ペンギン，ウミガメ`の6種のみ対応．

## Requirements
Xcode:                     10.2.1(10E1001)  
python:                    2.7  
Tensorflow:                1.1.0  
Keras      :               2.0.4  
Coremltools :              0.4.0
