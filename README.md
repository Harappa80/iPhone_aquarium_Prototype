# iPhone_aquarium_Prototype
## Overview
水族館図鑑アプリ(iOS)のプロトタイプ.</br>
iPhoneにて撮影した写真から対象の水族館生物を認識し，その生物に関する情報を表示．</br>
プロトタイプでは`イルカ，クラゲ，カクレクマノミ，カワウソ，ペンギン，ウミガメ`の6種のみ対応．</br>

## Requirements
Xcode:                     10.2.1(10E1001)  
python:                    2.7  
Tensorflow:                1.1.0  
Keras      :               2.0.4  
Coremltools :              0.4.0
